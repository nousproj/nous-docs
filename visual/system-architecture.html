Critical Design Insight — Container orchestration patterns don't map 1:1 to cognitive workloads. Agents are probabilistic (not deterministic), have complex cognitive state (not externalized), require semantic health evaluation (not binary TCP probes), and need multi-objective scheduling (cost × quality × latency, not just CPU/memory bin-packing). Nous is NOT a Kubernetes operator — it is a standalone control plane, runtime-agnostic across ECS, Lambda, VMs, and edge devices.
Nous Resources vs Kubernetes
AgentDefinition ← Deployment
Declares desired state of an agent type: model, tools, scaling policy, health gates, memory config.
AgentTask ← Job
A unit of work. Input prompt, budget, timeout, retry policy. Assigned to an agent instance by the scheduler.
AgentInstance ← Pod
Running instance of an agent. Controller-managed (not user-created). Tracks cognitive state, metrics, phase.
